Final Stats — soul globals update live with the soul levels above
Final = HP Total × (1 + Base HP Total % ÷ 100) × (1 + Global% ÷ 100) — the three in-game numbers that make the final (same formula as the Stat Calculator).
• HP Total — flat total (Initial Attributes + Equip + Adventure + Enchant + Avian … all fold into this).
• Base HP Total % — the big base multiplier row from the game (e.g. 2308.6k%).
• Global % (minus all from souls) — your Global % with the soul share removed. The soul gain lands here: the tool adds the HP/ATK/DEF soul’s global back in to get Global % (total), so Final moves live as those soul levels change.
Inputs accept k/m/b/t suffixes; hover a Final for the exact number. Remaining turns red if the set costs more essence than you own.
